# Sales & Revenue Analysis Dashboard

## Overview
The **Sales & Revenue Analysis Dashboard** is an interactive data visualization project developed using Microsoft Excel. This dashboard helps analyze sales performance, revenue trends, and top-performing products using Excel charts, KPIs, filters, and slicers.

The project uses datasets imported from:
- `sales_data.xlsx`
- `dashboard.xlsx`
- `.csv` sales dataset

The main objective of this project is to generate meaningful business insights through data analysis and visualization.

---

# Features

## Data Import
- Import data from Excel (`.xlsx`) files
- Import data from CSV (`.csv`) files
- Structured dataset support for future database integration

## KPI Tracking
The dashboard displays important Key Performance Indicators (KPIs) such as:
- Total Sales
- Total Orders
- Average Sales
- Revenue Performance

## Interactive Visualizations
- Revenue Trend Line Chart
- Sales Comparison Bar Chart
- Top Performing Products Analysis
- Interactive Filters and Slicers

## Business Insights
- Analyze monthly and yearly sales trends
- Identify high-performing products
- Compare sales performance across categories
- Generate business insights for decision-making

# Dashboard Components

## KPI Cards
The dashboard includes KPI cards displaying:
- Total Sales
- Total Orders
- Average Sales

## Revenue Trend Analysis
A line chart is used to visualize:
- Revenue growth over time
- Monthly sales trends

## Sales Comparison

- Product-wise sales performance
- Top-performing products

## Filters & Slicers
Interactive slicers help users:
- Filter data dynamically
- Analyze specific categories or periods
- Improve user interaction with the dashboard
